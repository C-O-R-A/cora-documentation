ROS 2 Package Reference
=======================

Generated by ``rosdoc2`` from source packages in the monorepo.
Output is written to ``_build/html/api/ros2/<package>/`` — no external links.

Run ``make ros2docs`` to (re)build all packages, or target a single one:

.. code-block:: bash

   make _ros2_cora_moveit
   make _ros2_cora_msgs

----
