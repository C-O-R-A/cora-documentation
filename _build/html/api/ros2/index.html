ROS 2 Package Reference¶

Generated by rosdoc2 from source packages in the monorepo. Output is written to _build/html/api/ros2/<package>/ — no external links.

Run make ros2docs to (re)build all packages, or target a single one:

make _ros2_cora_moveit
make _ros2_cora_msgs

Packages¶

cora_common — shared across all workspaces

cora_bringup

Launch files for bringing up the full CORA system

cora_codi

ROS 2 wrapper node for the codi Python SDK

cora_description

URDF and mesh files for the CORA arm

cora_gripper_1_description

URDF and mesh files for gripper variant 1

cora_moveit_config

MoveIt 2 SRDF, kinematics, and planning pipeline configuration

cora_moveit_cpp

MoveIt 2 C++ motion planning nodes

cora_moveit

MoveIt 2 integration and planning interface

cora_msgs

Custom ROS 2 message and service definitions

cora_vision

Vision pipeline: ArUco detection, pose estimation

cora_vision_msgs

Custom message definitions for the vision pipeline

cora_desktop — simulation and desktop tools

cora_gazebo

Gazebo Harmonic world, spawn, and controller launch files

cora_robot — hardware-specific packages

odrive_ros2_control

ros2_control hardware interface for ODrive motor controllers
